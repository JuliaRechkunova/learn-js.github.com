About javascript tutoring:

Leading up to the v1.0.0 release, I'm offering one-on-one javascript tutoring as a bonus for purchasing the book early. This will help you get questions answered about any javascript/html/css project you're working on.

It will also help me get a sense of what content should be added and revised in the book. I'll craft the remaining chapters based on the conversations in these tutoring sessions.

The tutoring sessions will be most helpful for folks getting started with javascript. If you're already a javascript developer you might want to get the book on its own via leanpub or gumroad.

    Learn.js will introduce you to:

  • Javascript syntax, functions, data types, and programming patterns.
  • How to find and use Javascript libraries.
  • Using javascript/jquery to manipulate html elements.
  • Using javascript for server-side coding.
  • Using developer tools like Git, GitHub, Chrome Dev Tools, Bower, npm, and Grunt.
  • How to build a wide range of projects with javascript, from the server to the client, from game development to data analysis.
