introduction to javascript

Learn the basics of Javascript, js libraries, and developer tools.

Workshop overview:

  • Learn Javascript syntax, functions, data types, and programming patterns.
  • Learn to find and use Javascript libraries.
  • Get familiar with developer tools like Git, GitHub, Chrome Dev Tools, Bower, npm, and Grunt.
  • Work on projects where you make interactive maps and charts, access data from the APIs of services like Twitter or data.seattle.gov.
  • Create interactive graphics that form the building blocks of Javascript game development.
